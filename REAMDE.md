## Тестирование:

Для тестирования использовался фреймворк pytest.
Если у вас его нет, то скопируйте команду для установки.
```
poetry add --group dev pytest
```
Для проверки покрытия тестами, использовалась библиотека pytest-cov.
Если ее нет, то скопируйте команду для установки.
```
poetry add --group dev pytest-cov
```
Для проверки покрытия тестами используйте команду.
```
poetry run pytest --cov
```
Для формирования отчета об покрытии тестами в формате HTML, используйте команду.
```
pytest --cov=src --cov-report=html
```
где src - пакет с модулями, которые тестируются.
Отчет будет находиться в htmlcov/index.html
